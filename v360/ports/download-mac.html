  • Step 1: download

    Click on the below to start the download system. Depending on your browser you may need to right-click so that the file is saved to disk and not displayed. Problems? Check our list of additional mirrors .

    NetHack-360-mac-Term02.pkg   (about 2M)   (MD5 sum: 9669d83541f6dfb65700aaad716b9803)
    NetHack 3.6.0 for Intel Macintosh running MacOS 10.7 Lion and later. This is a very traditional NetHack that runs in text mode from the command line or from Finder.

     

  • Step 2: Mount

    If your browser did not mount the image, double-click on the downloaded file and MacOS will mount the image and start the Installer.

  • Step 3: Install

    If the Installer has not started, double-click on the package file. The MacOS X Installer program will start and ask for the root password. The game will be installed under /usr/local/games and /Applications.

    If you will be running NetHack from the shell, edit the $PATH variable for your shell to include /usr/local/games/bin.

  • Step 4a: Run from the Command Line

    Run nethack in a Terminal window. The man pages are in /usr/games/man and the Guidebook is in /usr/games/doc.

  • Step 4b.: Run from Finder

    The following applications are in the Applications folder:
    • NetHackTerm       the game
    • NetHackGuidebook  the documentation
    • NetHackRecover    a program for recovering after a game or system crash
