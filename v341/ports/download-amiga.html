  • Step 1: download.

    Click on the below to start the download system. Depending on your browser you may need to right-click so that the file is saved to disk and not displayed. Problems? Check our list of additional mirrors .

  • Step 2: unpack.

    Change directory to a new directory and then unpack with:
    lha x nh341XXX.lha
    where XXX matches the file downloaded in step 1.

  • Step 3: install.

    Read the Guidebook for a general description of what NetHack is and how to play.
    Edit nethack.cnf to adjust your settings preferences.
    Assign NetHack: to the NetHack directory created in step 2.
    Add NetHack: to your path.

  • Step 4: run.

    Run nethack

    Refer to the file readme, the comments in nethack.cnf and the GuideBook and Install.ami within the lha-archive file for further instructions.
