--- layout: legacy title: Atari Downloads legacy: back_link: text: NetHack 3.3.1 Downloads url: ../downloads.html --- This binary runs on Ataris running TOS, MiNT or MagiC. This is a GEM (tiles) and TTY (plain text only) port of NetHack. It requires at least 2MB free memory and at least 3MB of free hard drive space.

  • Step 1: download.

    Click on the link below to download.

    Depending on your browser, you may need to right-click so that the file is saved to disk and not displayed.

    nh331gt.lzh(about 1.1M)
    NetHack3.3.1 for Atari

  • Step 2: unpack.

    These files can be unpacked with any standard lharc utility, which should be used when you have set your current directory to the location where you want to place NetHack.
    Make sure you create the subdirectories by applying the /x switch, i.e.: lha e /x nh331gt.lzh.

  • Step 3: install.

    Read the Guidebook for a general description of what NetHack is and how to play.
    Edit nethack.cnf to adjust your settings preferences. The default settings use a 16 colour tiled display.
    If you prefer the tty-display ensure that the line OPTIONS=windowtype=tty,rawio,BIOS is not commented with a #.

  • Step 4: run.

    Run nethack.prg.

    Refer to the file readme, the comments in nethack.cnf and the GuideBook within the lha-archive file for further instructions.
