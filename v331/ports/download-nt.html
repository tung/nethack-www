  • Step 1: download.

    Click on the link below to download.

    Depending on your browser, you may need to right-click so that the file is saved to disk and not displayed.

    nh331NTi.zip(about 1M)
    Intel WIN32 console binary distribution. This may be used on Intel machines running Windows NT 4.0 or later, and Intel machines running Windows 95. This is a TTY (plain text only) port of NetHack which uses the WIN32 console I/O subsystem - if you want the version that uses tiles (the pretty graphics pictures), download the msdos version instead.

  • Step 2: unpack.

    These files can be unpacked with any standard unzipping utility, which should be used when you have set your current directory to the location where you want to place NetHack.

  • Step 3: edit.

    Edit defaults.nh to adjust your settings preferences.

  • Step 4: run.

    Run Nethack.exe

    Refer to the file READ.ME within the zip file for further installation instructions.
