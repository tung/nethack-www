--- layout: legacy title: Macintosh Downloads legacy: back_link: text: NetHack 3.3.1 Downloads url: ../downloads.html --- This is the binary distribution of NetHack 3.3.1. NetHack is a fat binary, so it should run on both 68K and PPC machines. This package is freely distributable, see the file license for details.

  • Step 1: download.

    Click on the below to start the download system. Depending on your browser you may need to right-click so that the file is saved to disk and not displayed. Problems? Check our list of additional mirrors .

    nethack-331-fat.sit      
    NetHack3.3.1 for 68K and PPC Macintosh

    Or as a last resort:

    • nethack-331-fat.sit by FTP (North America)
    • nethack-331-fat.sit by FTP (Europe)

     

  • Step 2: unpack.

    Unpack the archive downloaded in step 1 using Stuffit Expander or a similar utility.

  • Step 3: install.

    Read the Guidebook for a general description of what NetHack is and how to play. Unfortunately, the Guidebook is > 32K, so you will need something other than SimpleText to view it. Edit the file NetHack Defaults in this directory to modify startup options. Feel free to turn off News when you get tired of seeing its message when you start up.

    If you have an older Mac with limited memory, you may need to decrease the memory allocated to the game. Select the NetHack application program in the Finder, and choose Get Info from the File menu. In the "Memory Requirements" section of the resulting window, halve the numbers which are displayed, then close the Info window.
