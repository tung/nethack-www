  • Step 2: unpack.

    • 2a. If you have a previous version of NetHack installed, you will need to uninstall the old version before installing the new version (the installer won't upgrade 3.4.2 in place since they have different application names: "Nethack 3.4.3 For PocketPC" vs "Nethack 3.4.2 For PocketPC").
      After uninstalling 3.4.2, you can copy any saved games manually. The default installation folder for 3.4.2 is \Program Files\NetHack-342. Default installation folder for 3.4.3 is \Program Files\NetHack-343
    • 2b. Run the installer package and follow on-screen instructions.

  • Step 3: run.

    A shortcut to NetHack will be created in the Games folder on the device.

    Installer information

    Nethack for Windows CE installation packages were created using EZSetup utility by Scott Ludwig available at http://www.eskimo.com/~scottlu/win/index.html
