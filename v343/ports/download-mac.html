--- layout: legacy title: Macintosh Downloads legacy: back_link: text: NetHack 3.4.3 Downloads url: ../downloads.html --- There are several binaries available for the Mac; you only need one of the following packages.
None of these binaries will run on 68K-based Macs. Only the first binary below will run under OS X Lion.
  • MacOS X Terminal Port for Lion
    • OS: MacOS X 10.7 Lion
    • Map: text
    • UI: Terminal (or via telnet or ssh)
    • Notes:
      Not compatible with previous save files. This is a fresh install and will clear everything (save files, high scores, etc).
  • Carbon Port
    • OS: MacOS 8.1 or later, MacOS 9, MacOS X (through 10.6 Snow Leopard)
    • Map: text
    • UI: GUI
    • Notes:
      Replaces previous Mac "Classic" port.
      Compatible with save files from nethack-341.sit.
  • MacOS X Tiles Port
    • OS: MacOS X (through 10.6 Snow Leopard)
    • Map: Tiles (graphics)
    • UI: GUI
    • Notes:
      Also known as the "Mac Qt" port.
      Compatible with save files from current and prior Terminal ports (e.g., nethack-342-mac-term.pkg.sit, nethack-343-mac-term.pkg.sit).
  • MacOS X Terminal Port
    • OS: MacOS X (through 10.6 Snow Leopard)
    • Map: text
    • UI: Terminal (or via telnet or ssh)
    • Notes:
      Compatible with save files from prior Terminal ports (e.g., nethack-342-mac-term.pkg.sit) and current Qt and Terminal ports (nethack-343-2-macosx-qt.tgz, nethack-343-mac-term.pkg.sit).

  • Step 1: download (MacOS X Terminal Port for Lion)

    Click on the below to start the download system. Depending on your browser you may need to right-click so that the file is saved to disk and not displayed. Problems? Check our list of additional mirrors .

    NetHack-343-Lion-mac-Term-1.dmg   (about 1.4M)   (MD5 sum: b700ac166488dbbd810caccdd1ba1667)
    NetHack 3.4.3 for Intel Macintosh running MacOS 10.7 Lion. This version is a very traditional NetHack that runs in text mode from the command line or from Finder.

     

  • Step 2: Mount

    If your browser did not mount the image, double-click on the downloaded file and MacOS will mount the image and start the Installer.

  • Step 3: Install

    If the Installer has not started, double-click on the package file. The MacOS X Installer program will start and ask for the root password. The game will be installed under /usr/games and /Applications.

    If you will be running NetHack from the shell, edit the $PATH variable for your shell to include /usr/games/bin.

  • Step 4a: Run from the Command Line

    Run nethack in a Terminal window. The man pages are in /usr/games/man and the Guidebook is in /usr/games/doc.

  • Step 4b.: Run from Finder

    The following applications are in the Applications folder:
    • NetHackTerm       the game
    • NetHackGuidebook  the documentation
    • NetHackRecover    a program for recovering after a game or system crash

  • Step 1: download (MacOS Carbon Port)

    Click on the below to start the download system. Depending on your browser you may need to right-click so that the file is saved to disk and not displayed. Problems? Check our list of additional mirrors .

    nethack-343-mac-carbon.sit   (about 1.1M)   (MD5 sum: 7125425d3644ad962a330db45fa88473)
    NetHack 3.4.3 for PPC Macintosh running MacOS 8.5, 9.x, or 10.x. This version uses Carbon to provide a GUI interface and a text map.

     

  • Step 2: unpack.

    Unpack the archive downloaded in step 1 using Stuffit Expander or a similar utility. (Your browser may have already done this for you.)

  • Step 3: run.

    Click the NetHack icon to begin the game.
    Please read the Guidebook for a general description of what NetHack is and how to play. Unfortunately, the Guidebook is > 32K, so you will need something other than SimpleText to view it on MacOS 9 and earlier. You may edit the "NetHack Defaults" file to modify startup options. Feel free to turn off News when you get tired of seeing its message when you start up.

  • Step 1: download (MacOS X with tiles)

    Click on the below to start the download system. Depending on your browser you may need to right-click so that the file is saved to disk and not displayed. Problems? Check our list of additional mirrors .

    NetHack-343-3-macosx-qt.dmg   (about 5.7M)   (MD5 sum: 11b6fd2b592f690a3591a7e30c1248ed)
    NetHack 3.4.3 for PPC Macintosh running MacOS 10.2 or later. This version uses Qt to provide a tiles (graphical) display. It has been tested with MacOS 10.2 through 10.5.

    Note: there is a known bug with this package: The key combination cmd-Z will cause the game to lock up and you will need to use Force Quit to stop it. See the documentation or this page for recovery information.

     

  • Step 2: Mount.

    If your browser did not mount the image, double-click on the downloaded file and MacOS will mount the image and pop up a window.

  • Step 3: install.

    Drag the NetHackQt application and the Documentation folder somewhere convenient.

    Read the Guidebook for a general description of what NetHack is and how to play.
