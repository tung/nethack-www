  • Step 1: download.

    Official NetHack 3.2.3 binaries for Intel 386 compatible machines running DOS. In general, the protected mode binary is recommended, except where a machine has very limited memory resources, or does not have at least a '386 processor.

    Click on the appropriate link below to download. Depending on your browser, you may need to right-click so that the file is saved to disk and not displayed.

    nh323pm.zip
    This contains the protected-mode version of NetHack built with djgpp V2. It requires at least 3MB extended memory and at least 2.8MB of free hard drive space to unzip. It requires DPMI to run, and includes CWSDPMI for those that don't already have DPMI. A '386 or greater processor is needed.

  • Step 2: unpack.

    These files can be unpacked with any standard unzipping utility, which should be used when you have set your current directory to the location where you want to place NetHack.

  • Step 3: configure.

    Edit the NetHack.cnf to your liking. The default settings use ascii graphics. If you prefer a 16 colour tiled display, ensure that the line OPTIONS=video:autodetect is not commented with a #.

  • Step 4: run.

    Run NetHack.exe.

    See the file README in the top level directory unpacked in Step 2 for additional installation instructions.
