--- layout: legacy title: OS/2 Downloads legacy: back_link: text: NetHack 3.2.3 Downloads url: downloads.html --- The official NetHack 3.2.3 binary for Intel 386 compatible machines running OS/2.

  • Step 1: download.

    Click on the link below to download.

    Depending on your browser, you may need to right-click so that the file is saved to disk and not displayed.

    nh323os2.zip
    This file packaged with InfoZip 5.x and contains a 32bit binary that should run on any version of OS/2, 2.0 or later (although it has only been tested on 4.0). It was built using EMX 0.9d and was statically linked and, thus, does not require any additional libraries to run.

  • Step 2: unpack.

    NetHack is shipped in a single archive created with zip - unpack it in an empty directory.

  • Step 3: install.

    See the README unpacked in step 2 for additional instructions.
