  • Step 2: unpack.


    NetHack is shipped in a single archive created with tar and then shrunk with gzip. To unpack under Unix, the following commands should be executed in an empty directory: 
    	gzcat nethack-322.tgz > nethack-3.2.2.tar
	tar xf nethack-3.2.2.tar
    The gzip package (containing gzip, gunzip, and gzcat) may be obtained from the Free Software Foundation. Versions of tar and patch are also available from the same site.

    Apply the patch using the command: 

    	patch < nethack-3.2.3.patch

  • Step 3: build.

    See the file README in the top level directory unpacked in Step 2 for build instructions for the various operating systems.

    If you want to keep your save and bones files from 3.2.2, edit patchlevel.h and define VERSION_COMPATIBILITY.
