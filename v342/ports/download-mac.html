  • MacOS X with tiles

  • Step 1: download (MacOS X with tiles)

    Click on the below to start the download system. Depending on your browser you may need to right-click so that the file is saved to disk and not displayed. Problems? Check our list of additional mirrors .

    NetHack-3.4.2-2-MacOSX.tar.gz   (4.6M bytes)   (MD5 sum: d70aea1f87fe5249f99789c4edcafb02)
    NetHack 3.4.2 for PPC Macintosh running MacOS 10.x. This version uses Qt to provide a tiles (graphical) display.

  • Step 2: unpack.

    Double-click on the icon to unpack NetHack.

  • Step 3: install.

    You probably want to drag the resulting "NetHack" icon somewhere convenient.

    Read the Guidebook for a general description of what NetHack is and how to play.

    • Step 1: download (MacOS X traditional)

      Click on the below to start the download system. Depending on your browser you may need to right-click so that the file is saved to disk and not displayed. Problems? Check our list of additional mirrors .

      nethack-342-mac-term.pkg.sit   (1.7M bytes)   (MD5 sum: b9b8c5b1ec5845960998656da378e989)
      NetHack 3.4.2 for PPC Macintosh running MacOS 10.x. This version is a very traditional NetHack that runs in text mode from the command line.

    • Step 2: Unpack

      Unstuff with Stuffit Expander.

    • Step 3: Install

      Double-click on package file. The MacOS X Installer program will start and ask for the root password. The game will be installed under /usr/games.

      Edit your $PATH variable for your shell to include /usr/games.

      Read the README file. Note that unlike other 3.4.2 binary distributions, the save and bones files from this distribution are not backwards compatible.

    • Step 4: Run

      Run nethack in a terminal window.
