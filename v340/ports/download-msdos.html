  • Step 1: download.

    Official NetHack 3.4.0 binaries for Intel 386 compatible machines running DOS or Windows 3.1. In general we recommend the Windows version if your machine can run it.
    Note that it is no longer necessary to use this binary to get a graphical interface. The Windows version now supports that functionality.

    Click on the appropriate link below to download from the site closest to you. Depending on your browser (and the link you select), you may need to right-click so that the file is saved to disk and not displayed. Problems? Check our list of additional mirrors .

    nh340dos.zip   (about 1.1M)   (MD5 sum: c0e6ff55c431ff1c4210a473517bf058)
    This contains the protected-mode version of NetHack built with djgpp V2. It requires at least 3MB extended memory and at least 2.8MB of free hard drive space to unzip. A '386 or greater processor is needed. (about 1.1M)

    It requires DPMI to run. If you want to run it on plain DOS you can get CWSDPMI , unpack it, and put the CWSDPMI executable in your NetHack directory.

  • Step 2: unpack.

    These files can be unpacked with any standard unzipping utility, which should be used when you have set your current directory to the location where you want to place NetHack.

  • Step 3: configure.

    Edit the defaults.nh to your liking. The default settings use ASCII graphics. If you prefer a 16 colour tiled display, ensure that the line OPTIONS=video:autodetect is not commented with a #.

  • Step 4: run.

    Run NetHack.exe.

    See the file README in the top level directory unpacked in Step 2 for additional installation instructions.
